# reactvite README

> **Note:** This README is based on analysis of the uploaded React + Vite project.

## Project Overview
This project is a React + Vite frontend for the HRMS Training Management System.

## Features
- React UI
- Vite development server
- Tailwind CSS styling
- Axios API integration
- Authentication flow
- Training management UI

## Tech Stack
- React
- Vite
- Tailwind CSS
- Axios
- Framer Motion
- Lucide React

## Prerequisites
- Node.js 18+
- npm

## Installation
```bash
npm install
npm run dev
```

## Environment Variables
Create `.env`:
```env
VITE_API_BASE_URL=http://localhost:8080/api
```

## Build
```bash
npm run build
npm run preview
```

## API
Configured to communicate with the Spring Boot backend.

## Deployment
Build using `npm run build` and deploy the `dist` folder.

## Troubleshooting
- Verify backend is running.
- Verify `.env`.
- Reinstall dependencies if needed.
